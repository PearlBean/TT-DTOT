# Speed Project

Một dự án nhỏ chứa các file Arduino và một server Node.js để minh họa.

## Mô tả

Thư mục này bao gồm:

- `main.ino` - mã nguồn Arduino (chưa mô tả chi tiết).
- `server.js` - server Node.js (có thể dùng để giao tiếp hoặc chạy API nhỏ).
- `package.json` - manifest Node.js.

## Yêu cầu

- Node.js (phiên bản >= 12). Cài đặt Node từ https://nodejs.org/
- Nếu cần, chạy `npm install` để cài phụ thuộc (nếu có trong `package.json`).

## Cách chạy server

1. Mở terminal tại thư mục dự án (`c:\Users\ADMIN\Desktop\speed`).
2. Cài đặt phụ thuộc (nếu cần):

   npm install

3. Chạy server:

   node server.js
